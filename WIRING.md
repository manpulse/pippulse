## Pin Summary

| ESP32 Pin | Connected To |
|-----------|--------------|
| 3.3V | All VCC/VIN |
| GND | All GND |
| GPIO4 | DS18B20 DATA |
| GPIO21 | MAX30102 SDA |
| GPIO22 | MAX30102 SCL |
| GPIO25 | AD8232 LO+ |
| GPIO26 | AD8232 LO- |
| GPIO34 | AD8232 OUTPUT |
